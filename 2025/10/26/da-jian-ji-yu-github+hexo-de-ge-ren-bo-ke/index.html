搭建基于github+hexo的个人博客

搭建过程：
1、注册 GitHub 账号
2、安装 node.js
3、安装 git
4、配置 GitHub ssh
5、在本地新建一个文件夹用于存放 hexo 的源文件，并根据 hexo 文档进行安装

6、在_config.yml 中进行配置

7、最后执行hexo g和hexo d即可搭建完成

搭建中遇到的问题就是配置的时候把 repo 写成 rope，最后还是 ai 发现的问题
